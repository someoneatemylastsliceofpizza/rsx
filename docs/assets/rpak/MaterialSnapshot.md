# Material Snapshot (`msnp`)
<sup>This asset type does not exist in Titanfall 2</sup>

Material snapshot assets contain data that is used to create Direct3D states and descriptors for customised rendering of any dependent material assets.

This data includes:
- Blend states (D3D11_RENDER_TARGET_BLEND_DESC)
- Depth stencil flags (D3D11_DEPTH_STENCIL_DESC)
- Rasterizer flags (D3D11_RASTERIZER_DESC)

For more information about each of thes values, see the [Material](./Material.md) asset page.

This asset type does not exist in Titanfall 2
