# Particle Effect (`efct`)
<sup>This asset type does not exist in Titanfall 2</sup>


This asset is effectively an rpak-ified version of a [PCF](https://developer.valvesoftware.com/wiki/PCF) file, which is ultimately just an extension of [DMX](https://developer.valvesoftware.com/wiki/DMX). These define particle effects for use throughout the engine.

In Titanfall 2, particle effects are instead stored as .pcf files inside of [VPK](https://developer.valvesoftware.com/wiki/VPK_(file_format)) container files.


Not yet implemented.
