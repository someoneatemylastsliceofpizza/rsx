# Animation Rig (`arig`)

Lightweight version of a [Model](./Model.md) file containing only* bones, attachments, hitboxes, and IK data.

This asset uses [animation sequence](./AnimationSequence.md) assets to allow animations to be shared by models.

Animation Rigs can be used by multiple models, as long as the bones are compatible with that of the model.

(* there may be some other data that I've missed)
