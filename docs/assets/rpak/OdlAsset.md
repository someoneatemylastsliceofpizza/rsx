# ODL Asset (`odla`)
<sup>This asset type does not exist in Titanfall 2</sup>

On-Demand Loading Asset.

This asset acts as a wrapper for any asset that is stored in an external pak (i.e., one that is not constantly loaded) that can be loaded on-the-fly.

ODLA assets point to both the original asset and an "ODL Pak" asset, allowing the game to locate the pakfile that contains the original asset.

Not yet implemented.
